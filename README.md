# Laura Hybrid Coach

App personal de entrenamiento híbrido para fuerza, running, Hyrox y progreso.

## Cómo abrir en VS Code

```bash
npm install
npm run dev
```

## Cómo publicar en GitHub Pages

1. Sube este proyecto al repositorio `Laura-hybrid-coach`.
2. En terminal:

```bash
npm install
npm run build
npm run deploy
```

3. En GitHub ve a:
Settings > Pages > Source: Deploy from branch > Branch: gh-pages > Save.

## Cómo instalar en móvil

iPhone:
Safari > Compartir > Añadir a pantalla de inicio.

Android:
Chrome > Menú > Añadir a pantalla de inicio.
